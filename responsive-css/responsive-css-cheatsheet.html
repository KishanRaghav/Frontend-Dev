Responsive CSS Cheat Sheet

Filters
filter: grayscale(100%); — Use on images to stylize; remove on hover for effect.
Media Queries
@media (max-width: 600px) { ... } — Change layout at breakpoints.
Responsive Grid
grid-template-columns:repeat(auto-fit,minmax(200px,1fr));
Flexbox
display:flex;justify-content:center; — Great for alignment and stacking.
Typography
font-size:clamp(1rem,2vw,1.5rem); — Scales text across viewports.
Transitions
transition:all .3s ease; — Smooth state changes on hover/focus.
