Responsive Design Principles

Fluid Layouts

Designs that scale with viewport using flexible grids and percentages.

.container{max-width:100%;padding:1rem;}

Flexible Media

Images & media using max-width:100% to fit containers.

img{max-width:100%;height:auto;}

Breakpoints

Use media queries to alter layout at key widths.

@media (max-width:600px){...}
