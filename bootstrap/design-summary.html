Responsive Design Principles

Fluid Layouts

Use percentage or flexible units to allow content to resize based on screen width.

width: 90%;
max-width: 1200px;

Flexible Media

Images, videos, and media should scale to fit their container without overflow.

img {
  max-width: 100%;
  height: auto;
}

Breakpoints

Define CSS rules that apply at specific screen widths to change layout or style.

@media (max-width: 600px) {
  .grid { grid-template-columns: 1fr; }
}
